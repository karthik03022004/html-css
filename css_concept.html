CSS(cascading Style sheets)

Cascading means flow of execution,style means adding designs.

  1. Main purpose is to create a responsive web design and animations.

  2. CSS is used for to make webpages attractive by adding color to the text and at background also.

To implement CSS we have 3 methods:

  • Inline:we implement our styling at element in a single line.
  • Internal: At head tag we crate a style tag and implement our styles.
  • External: we create a new a new file with extension(.css) and there we implememt our styling.
NOTE: When we apply styling in 3 methods inline,internal,external to give priroity we use (!important) beside color so, it thinks it is mandatory.

SELECTORS,PROPERTIES,VALUES:

  • Selectors are nothing but It gives stylings to the targeted elements only.

  • Properties describes where the color be applied like background or the text etc.

  • color describes what type of colors we are using on text background etc.. ,For example light-blue,dark violetEtc..

About SELECTORS

There are 5 types of selectors:

  1. Simple
  2. Combinators
  3. Attributes
  4. pseudoclass
  5. Pesudoelements

again in simple selectors we have 5 types:

Simple Selectors:

  • id(represented as #)->it uniquely apply stylings
  • class(represented as .)-> to apply styles to same elements we use grouping.
  • element -> we mention element as selector
  • grouping(mentioned by ,)-> we take multiple selectors
  • universal(*)-> applies colors to whole body

Priority:id>class>ele=grouping>universal
